# PoCMLOps
Ein kleiner Prototyp um zu zeigen, dass ich Grundlagen in MLOps (Tracking/Model repo via. MLFlow, Monitoring via. Prometheis/Grafana und ETL) behersche

Die Stellt nicht meine gewöhnliche Arbeitsqualität dar, ich habe den Code dieses Repos recht schnell zusammen geworfen. Nichts destotroz ist es funktional und kann über das `run_docker_compose.sh` gestartet werden.

Um ein der api service wird zunächst Fehlschlagen, da noch kein Model im repository registiert ist. Ein sehr simples demo model, da ich hier MLOps, nicht DL zeigen möchte, ist im Github repo vorhanden. Ebenso Beispieldaten und ein `config_dvc.sh` Skript um nach dessen Initalisierung die Konfiguration vorzunehmen.
